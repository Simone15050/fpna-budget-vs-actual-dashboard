## Dashboard Preview
![Dashboard Preview](dashboard_preview.png)

## Project Overview
This project simulates a 12-month financial performance analysis for a business unit. The model compares budgeted and actual revenue and expenses, calculates monthly and annual variances, and presents key financial metrics through an executive-ready dashboard.

The goal of this project was to strengthen financial reporting, variance analysis, and structured dashboard design skills relevant to FP&A and financial analytics roles.

---

## Key Features

- 12-month Budget vs Actual analysis
- Revenue and expense variance calculations
- Net income and profit margin modeling
- Executive KPI dashboard
- Conditional formatting for favorable/unfavorable variances
- Clean, stakeholder-ready PDF reporting output

---

## Tools Used

- Microsoft Excel
- Financial modeling techniques
- Variance analysis
- KPI dashboard development
- Data visualization best practices

---

## Key Insights

- Actual revenue exceeded budget in 9 of 12 months.
- Total annual revenue variance was favorable at $130,000.
- Net income totaled $2.87M, resulting in a 38.47% profit margin.
- Expense variances remained moderate and controlled throughout the year.
